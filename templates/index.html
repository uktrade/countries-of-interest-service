/api/v1/get-company-countries-and-sectors-of-interest

Returns a list of company - country and sector of interest relationships for companies stored in Datahub. These companies have expressed interest in the country-sector combination. You can use the country, sector and source arguments to filter on country, sector and data source.

e.g.

  • /api/v1/get-company-countries-sectors-of-interest
  • /api/v1/get-company-countries-sectors-of-interest?country=CN
  • /api/v1/get-company-countries-sectors-of-interest?sector=Aerospace
  • /api/v1/get-company-countries-sectors-of-interest?source=omis
  • /api/v1/get-company-countries-sectors-of-interest?country=CN&sector=Aerospace
  • /api/v1/get-company-countries-sectors-of-interest?company-id=d0af8e52-ff34-4088-98e3-d2d22cd250ae

/api/v1/get-company-countries-of-interest

Returns a list of company - country of interest relationships for companies stored in Datahub.

e.g.

  • /api/v1/get-company-countries-of-interest
  • /api/v1/get-company-countries-of-interest?country=CN
  • /api/v1/get-company-countries-of-interest?source=omis
  • /api/v1/get-company-countries-of-interest?country=CN&source=omis
  • /api/v1/get-company-countries-of-interest?company-id=d0af8e52-ff34-4088-98e3-d2d22cd250ae

/api/v1/get-company-export-countries

Returns a list of company-country export relationships for companies stored in Datahub.

e.g.

  • /api/v1/get-company-export-countries
  • /api/v1/get-company-export-countries?country=CN
  • /api/v1/get-company-export-countries?source=datahub_export_countries
  • /api/v1/get-company-export-countries?country=MY&source=datahub_export_countries
  • /api/v1/get-company-countries-of-interest?company-id=d0af8e52-ff34-4088-98e3-d2d22cd250ae

/api/v1/get-company-sectors-of-interest

Returns a list of company-sector of interest relationships.

e.g.

  • /api/v1/get-company-sectors-of-interest
  • /api/v1/get-company-sectors-of-interest?sector=Food
  • /api/v1/get-company-sectors-of-interest?source=omis
  • /api/v1/get-company-sectors-of-interest?sector=Aerospace&source=omis
  • /api/v1/get-company-sectors-of-interest?company-id=d0af8e52-ff34-4088-98e3-d2d22cd250ae
