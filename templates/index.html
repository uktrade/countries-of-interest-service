API

/get-companies-affected-by-trade-barrier/<country>/<sector>

Returns a list of companies which are affected by a specified country and sector. Countries are specified using the ISO Alpha-2 codes which can be found here and sector segments can be found here.

e.g. /get-companies-affected-by-trade-barrier/AF/Advanced%20Engineering

/get-company-countries-and-sectors-of-interest

Returns a list of company - country and sector of interest relationships for companies stored in Datahub. These companies have expressed interest in the country-sector combination.

.e.g /get-company-countries-sectors-of-interest

/get-company-countries-of-interest

Returns a list of company - country of interest relationships for companies stored in Datahub.

.e.g /get-company-countries-of-interest

/get-company-export-countries

Returns a list of company-country export relationships for companies stored in Datahub.

.e.g /get-company-export-countries

/get-company-sectors-of-interest

Returns a list of company-sector of interest relationships.

.e.g /get-company-sectors-of-interest
